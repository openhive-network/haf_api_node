HAF pgBadger Config

If you're seeing these instructions, it means you have never run pgBadger.

You must enable pgBadger by adding the pgbadger profile to your COMPOSE_PROFILES= in your .env file, then running docker compose up -d again. This will configure the stack to run pgBadger once a day at UTC midnight. After it has run, it will replace this page with the pgBadger results.
